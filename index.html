Tools Components:



Beads


Upper Beads (Heaven Beads): Located above the horizontal bar (beam), each upper bead represents a value of 5.


Lower Beads (Earth Beads): Located below the horizontal bar, each lower bead represents a value of 1.


Columns:


The abacus consists of several vertical columns, each representing a digit in a numerical value


Horizontal Bar (Beam):


This bar separates the upper and lower beads. It is used as a reference point for counting.


Divider Line:


A vertical line that separates the two sets of beads, helping in visual organization.


Frame:


The outer structure that holds the columns and allows for easy manipulation of beads.


Carryover System:


There may be a carryover system to handle carrying over values to the next column when a specific threshold is reached.
